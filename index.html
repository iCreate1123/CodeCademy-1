CSS Color Reference
Cheat sheet

CSS supports various color formats to define colors. Each format has its own syntax and use cases. Here are some common color formats in CSS:

Name Colour format Description Example
Hexadecimal #RRGGBB or #RRGGBBAA (with an optional alpha channel for transparency) Represents a color using a combination of red, green, and blue values in hexadecimal format. #FF0000 for red.
RGB rgb(R, G, B) or rgba(R, G, B, A) (with an optional alpha channel for transparency) Represents a color using the red, green, and blue values in decimal format. rgb(255, 0, 0) for red.
HSL hsl(H, S%, L%) or hsla(H, S%, L%, A) (with an optional alpha channel for transparency) Represents a color using the hue, saturation, and lightness values. hsl(0, 100%, 50%) for red.
Named Colours red, blue, green, etc. Represents a color using a predefined name. Limited set of named colors available. Blue, green, white, purple
